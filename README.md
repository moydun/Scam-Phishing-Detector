##  1. О продукте
**ScamShield** — это супер-простая веб-платформа для мгновенной проверки подозрительных сообщений, интернет-ссылок и номеров телефонов. Продукт разработан специально с заботой о старшем поколении (аудитория 50+), чтобы защитить их от мобильного и интернет-мошенничества.

---

##  2. Проблема и Главная Фишка (Простой язык)
Большинство ИИ-детекторов выдают сложные технические термины вроде *«Обнаружен фишинговый URL»*, которые непонятны пожилым людям. 

**Главное правило ScamShield — говорить на языке пользователя:**
* *Неправильно:* «Обнаружен фишинговый URL»
* *Правильно:* **«Не переходите по этой ссылке. Мошенники могут украсть ваши деньги».**

---

##  3. Функционал MVP (Ничего лишнего)
Мы полностью убрали сложную регистрацию, авторизацию и личные кабинеты. Сайт состоит из **одной страницы** и трех интуитивных блоков:

1. ** Блок 1 — Ввод данных:** Одно большое поле, куда можно вставить подозрительный текст (например, сообщение о «выигрыше»), ссылку или номер телефона. Кнопка «Проверить» + спиннер на время загрузки.
2. ** Блок 2 — Понятный вердикт:**
   * **Цветовой маркер:** 🟢 Безопасно, 🟡 Подозрительно, 🔴 Опасно.
   * **Объяснение:** Текст простыми словами для пожилых людей.
   * **Технический блок (для продвинутых/внуков):** Скрытая плашка, где выводятся IP, страна сервера, дата создания домена и отчет из VirusTotal.
3. ** Блок 3 — Экстренная помощь:** Кнопка «Нужна помощь?», которая перенаправляет пользователя в **Telegram-бота** (замена сложному чату на сайте).

 **Локализация:** Полная поддержка переключения интерфейса и вердиктов на **Русский (RU)** и **Кыргызский (KG)** языки.

## 4. Техническая реализация команды FishingScam

### Backend (Python / FastAPI)
Реализован всего **один универсальный эндпоинт `POST /analyze`**, который принимает строку (текст/ссылка/номер) и под капотом делает:
1. **VirusTotal API:** Проверка ссылки по мировым антивирусным базам.
2. **WHOIS-запрос:** Определение IP, страны хостинга и даты регистрации домена (мошенники часто используют «свежие» сайты).
3. **AI Core (Gemini):** Запрос к ИИ для генерации понятного вердикта.

*Пример эталонного ответа бэкенда:*
```json
{
  "verdict": "Scam",
  "risk_level": "высокий",

  "virustotal": "найдено в 12 базах",
  "whois": "сервер в России, создан вчера"
}
